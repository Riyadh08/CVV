### *APIs*
5. *Which HTTP method is most appropriate for modifying job titles?*
   - *Answer:* PATCH

6. *Difference between PUT and PATCH methods in REST API?*
   - *Answer:* PUT updates the entire resource, while PATCH updates only specific fields of a resource.

7. *Significance of HTTP status code 429?*
   - *Answer:* It is used to inform the client that the server is not prepared to handle the request due to too many requests being made, indicating rate limiting or throttling.

8. *Key advantages of implementing rate limiting and throttling mechanisms?*
   - *Answer:* Ensuring that API endpoints are always available and responsive.

9. *What does the HTTP status code "410 Gone" indicate?*
   - *Answer:* The resource is no longer available at the requested URL, and it won't be available in the future.

10. *Primary difference between SOAP and RESTful APIs?*
    - *Answer:* SOAP is protocol-agnostic, while REST is based on HTTP.
